

# NewFoodQuickDelivery

A Java-based food delivery application that simulates a simple ordering and delivery management system. This project focuses on core Java programming, classes and objects, collections and basic input/output.

---

## Overview

The app allows a customer to place an order from a restaurant, calculates totals and delivery details, and outputs a formatted summary. It was built as a bootcamp project to practise clean Java code structure and problem solving.

---

## Features

- Capture customer and restaurant details
- Store menu items and customer order
- Calculate order total and delivery information
- Output a neatly formatted order summary
- Basic error checking on user input

---

## Tech Stack

- **Language:** Java
- **Concepts:** OOP (classes, objects, methods), arrays / collections, control flow
- **Tools:** JDK, any Java IDE (e.g. IntelliJ, Eclipse, VS Code)
